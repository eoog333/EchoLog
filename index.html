시간의 흐름으로 정리해 줍니다.

녹음 버튼을 누르고 편하게 오늘 있었던 일을 이야기하세요.
당신의 목소리는 인공지능(RTZR STT API)을 거치며
불필요한 추임새들이 정제됩니다.

그리고 발화된 시간 순서대로 아름답게 나뉘어,
가장 읽기 편안한 한 편의 타임라인 회고록으로 완성됩니다. 

git clone https://github.com/eoog333/EchoLog-Demo.git
cd EchoLog-Demo

# 1. 백엔드 실행 (터미널 1)
cd backend
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
uvicorn app.main:app --reload

# 2. 프론트엔드 실행 (새로운 터미널 2)
cd ../frontend
npm install
npm run dev
